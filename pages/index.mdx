# Overview

```tsx twoslash
import React from 'react'
import '@razorlabs/razorkit/style.css'
import { ConnectButton, WalletProvider } from '@razorlabs/razorkit'

const YourApp = () => {
  return <ConnectButton />
}

const App: React.FC = () => {
  return (
    <WalletProvider autoConnect={true}>
      <YourApp />
    </WalletProvider>
  )
}
```

# Features

Razor Kit supports all these features out-of-the-box:

- Easy to Install
- Custom Themes
- Built-in Themes
- Custom Wallet Lists
- Support for all Movement Wallets
- Custom Connect Button
